# FiveM-AdministrationPanel
An Administration Panel for FiveM Servers


### Features
* Warning
* Kicking
* Temporary Banning
* Permanent Banning
* Playtime Tracking
* Staff Tracking
* Trust Score System
* Permissions Based
* In-Game Commands
* Indepth Web-UI


### Requirements
* Web Server w/ PHP
* MySQL Database
* Steam API Key (https://steamcommunity.com/dev/apikey)
* Support for Cron Jobs
