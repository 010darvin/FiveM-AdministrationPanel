# FiveM-AdministrationPanel
An Administration Panel for FiveM Servers


### Features
* Warning
* Kicking
* Temporary Banning
* Permanent Banning
* Playtime Tracking
* Trust Score System
* Indepth Web-UI
* Indepth API
* Custom Server Action Buttons
* Some Statistics
* Multiple Servers
* First Joined
* Last Played
* Warnings/Kicks/Bans Logging

### Upcoming Features
* Staff Tracking
* Permissions Based
* In-Game Commands
* Unban Button
* Add/Remove Staff **Menu**
* Add/Remove Server **Menu**
* Installer

### Requirements
* Web Server w/ PHP
* MySQL Database
* Steam API Key (https://steamcommunity.com/dev/apikey)
* Support for Cron Jobs
